# Start Here

If you are continuing local Waveary development, use the local-only continuity layer first:

1. Use `waveary-continuity-guard`
2. Read `.waveary-continuity/PROJECT_STATE.md`
3. Read `.waveary-continuity/ACTIVE_TASKS.md`
4. Read `.waveary-continuity/docs/product-preferences.md`
5. Read `.waveary-continuity/docs/session-log.md`
6. Read `.waveary-continuity/docs/decision-log.md`
7. Read `.waveary-continuity/docs/workflow-rules.md`
8. Check `git status --short -b`

The `.waveary-continuity/` directory is local-only and ignored by git.
