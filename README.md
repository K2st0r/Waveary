# Project Waveary

## 回响之境 | Waveary

**一个开源数字生命陪伴框架。**  
**An open source digital life companion framework.**

Waveary 不试图创造更聪明的 AI。  
Waveary does not try to build a smarter AI.

它试图创造一个能够记住、理解、成长并长期陪伴用户的人生伙伴。  
It tries to build a life companion that can remember, understand, grow, and stay with people over time.

> 记忆优先于模型。  
> 关系优先于功能。  
> 陪伴优先于智能。
>
> Memory comes before model.  
> Relationship comes before features.  
> Companionship comes before intelligence.

**Slogan / 核心标语**

念念不忘，终有回响。  
What is remembered returns as an echo.

## Overview

Waveary 是一个面向长期陪伴场景的开源基础框架，用来为任意大模型补上四种关键连续性：

Waveary is an open source framework for long-term digital companionship. It adds four kinds of continuity to any large model:

- 记忆连续性 / Memory continuity
- 关系连续性 / Relationship continuity
- 人生时间轴连续性 / Life timeline continuity
- 情感陪伴连续性 / Emotional continuity

它不是一个“AI 女友”项目，也不是一个普通聊天机器人壳。  
It is not an "AI girlfriend" project, and it is not just another chatbot wrapper.

它更接近于一层数字陪伴运行时。  
It is closer to an operating layer for digital companionship.

## Why Waveary

大多数 AI 产品优化的是“更好的回答”。  
Most AI products optimize for better answers.

Waveary 优化的是“更好的延续”。  
Waveary optimizes for better continuity.

大多数角色产品依赖 prompt、设定和短期互动。  
Most character products depend on prompts, personas, and short-term interaction.

Waveary 把记忆、关系、时间轴、情绪和语音当作第一等系统能力。  
Waveary treats memory, relationship, timeline, emotion, and voice as first-class systems.

## Core Capabilities

Waveary is designed as a composable engine stack:

- **Waveary Memory Engine (`WME`)**
  - 长期记忆 / Long-term memory
- **Waveary Relationship Engine (`WRE`)**
  - 关系成长 / Relationship growth
- **Waveary Timeline Engine (`WTE`)**
  - 回忆系统与人生事件 / Recall and life-event timeline
- **Waveary Emotion Engine**
  - 情绪建模与情感适配 / Emotional modeling and adaptation
- **Waveary Voice Engine (`WVE`)**
  - 语音交互能力 / Voice interaction
