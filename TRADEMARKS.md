Commercial use of the official Waveary marks or official brand assets requires prior written permission.

## Reporting And Enforcement

If you see confusing or misleading use of the Waveary marks, keep evidence such as:

- URLs
- screenshots
- store listings
- repository links
- publication dates

For repository impersonation or misleading branding on GitHub, see GitHub's trademark policy:

- [GitHub Trademark Policy](https://docs.github.com/en/site-policy/content-removal-policies/github-trademark-policy)

## Practical Rights Preservation

For stronger real-world enforceability, especially in China, the rights holder should also:

- file trademark applications for `Waveary`
- file trademark applications for `回响之境`
- file trademark applications for the main logo/device mark
- preserve early evidence of creation and public use, including git history, design source files, domain registration, screenshots, and first public posts
- consider software copyright registration and artwork/logo copyright evidence preservation
