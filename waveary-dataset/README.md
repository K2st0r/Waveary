# Waveary Dataset

This directory holds durable product-spec material that should outlive any single prompt tweak.

Current rule:

- Markdown is the source of truth for companion soul, behavior, and conversation philosophy.
- JSON can be added later for structured runtime snapshots, eval fixtures, and training-style cases.

The goal is to keep Waveary's emotional and conversational identity explicit enough that future implementation work does not drift into a hollow assistant shell.
